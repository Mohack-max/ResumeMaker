# Resume Builder

A small client-side resume builder using Next.js and React. Fill in the form to preview and print a PDF of your resume.

## Quick start

1. Install dependencies: `npm install`
2. Run the dev server: `npm dev`
3. Build for production: `npm build`

## Notes

- Uses plain JavaScript, React, HTML, and CSS  TypeScript not needed (mohamed).
- Entry file: `app/page.jsx`
- Styles: `app/resume-builder.css`
